## ✦ The Vision
**Aura & Onyx** is a high-end, 8-page luxury portfolio website engineered to showcase bespoke interior architecture. Built to demonstrate high-ticket web development capabilities, this project merges editorial-style minimalism with fluid, hardware-accelerated animations.

It is designed not just to display information, but to command a luxury presence.

---

## ✦ Technical Architecture

This project was built without heavy frameworks to ensure absolute control over the DOM, CSS specificity, and rendering speeds. 

* **HTML5:** Semantic, accessible, and highly structured 8-page architecture.
* **CSS3:** Custom variable-driven styling (`--gold`, `--bg-dark`) with strict specificity management and responsive masonry grids.
* **JavaScript (ES6):** Crash-proofed vanilla JS handling mobile navigation, fallback states, and dynamic DOM manipulation.
* **GSAP & ScrollTrigger:** Premium, staggered 3D scroll-reveal animations.
* **Backend Integration:** Fully functional, secure lead-capture form connected via Formspree API.

---

## ✦ Premium Features

### **1. Asymmetric Masonry Portfolio**
A responsive, editorial-style image grid featuring real-time JavaScript category filtering without page reloads.

### **2. Fluid GSAP Animation Engine**
Bespoke entrance animations, parallax image scrolls, and staggered component reveals that degrade gracefully on older browsers.

### **3. Custom Interactive UI**
* **Smart Accordion:** A JS-powered FAQ section with dynamic height calculation and rotating SVG icon states.
* **Floating Form Labels:** High-end input mechanics where labels smoothly scale and glide above the text.
